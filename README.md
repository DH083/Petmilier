# Petmilier
반려동물 관리 어플리케이션

# 🐾 프로젝트 소개
반려동물을 키우는 인구가 증가하고 있지만, 아직 여러 동물을 대상으로한 어플은 없다.
동물의 종과 상관없이 반려동물을 키우는 사용자들이 서로 정보를 공유할 수 있는 프로젝트를 목표로 진행하였다.

# 📅 개발 기간
2022.08.29 ~ 2022.11.18
